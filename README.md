# CODSOFT_3
# Iris Flower Classification

## Objective
Build a machine learning model that classifies Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

using sepal and petal measurements.

## Dataset Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Visualization
4. Data Preprocessing
5. Model Training
6. Model Evaluation
7. Prediction

## Results
The model successfully classifies Iris flowers into their respective species with high accuracy.
